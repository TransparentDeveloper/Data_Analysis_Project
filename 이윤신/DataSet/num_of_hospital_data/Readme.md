+ 2012.csv :
    + 2012 년 행정구역 별 병원 수 
        + citys                 :   행정구역
        + general_hospits_num	:   종합병원 수
        + general_hospits_bed	:   종합병원 침상 수
        + hospits_num	        :   일반병원 수           
        + hospits_bed	        :   일반병원 침상 수
        + affiliated_hospits_num:   부속병원 수
        + affiliated_hospits_bed:   부속병원 침상 수
